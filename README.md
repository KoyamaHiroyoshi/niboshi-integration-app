## WEBアプリ実行
- npm install -g @angular/cli
- cd niboshi-web-app
- npm install
- ng serve

## API起動
### local起動Ver
- niboshi-web-api配下の.env.template を.envとしてコピー
- Spring Boot Dahsboardからniboshi-interfacesを起動

### Docker起動Ver
- niboshi-web-api配下の.env.template を.envとしてコピー
- VSCodeタスクから「up-app-database」を実行
